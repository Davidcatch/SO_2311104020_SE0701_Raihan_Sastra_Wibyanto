## Dasar Teori

Virtual Box adalah perangkat lunak virtualisasi dari Oracle, virtual box sendiri bekerja dengan cara menciptakan lingkungan virtual yang menyerupai perangkat keras komputer.

Xinu OS merupakan sistem operasi yang dirancang khusus untuk perangkat embedded. Karena itu, proses pengembangannya menggunakan pendekatan cross-development.

Ubuntu adalah sebuah sistem operasi dan distribusi Linux berbasis Debian yang gratis dan open source. Sistem operasi ini dibangun dengan menggunakan infrastruktur Debian dan terdiri dari server, desktop dan sistem operasi Linux.

Sourcetrail adalah penjelajah kode sumber gratis dan terbukan untuk Gnu / linux, windows dan macOS
