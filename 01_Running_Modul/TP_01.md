## Dasar Teori

Virtual Box adalah perangkat lunak virtualisasi dari Oracle, virtual box sendiri bekerja dengan cara menciptakan lingkungan virtual yang menyerupai perangkat keras komputer.

## Guided

Langkah - langkah menjalankan backend dan development system atau server

1. Install virtual box di https://www.virtualbox.org/wiki/Download_Old_Builds_6_1

2. Pilih sesuai perangkat nya masing - masing misal (windows)

3. Jika sudah diinstall, jalankan virtual box nya ikuti prosesnya hingga selesai.

4. jika sudah terinstal, jalankan lagi virtual box nya, setelah terbuka di menu utama terdapat pada pojok kiri atas bernama berkas atau file, lalu pilih import dan select file berkestensi ova.

5. jika sudah dipilih lalu lanjut.

6. lanjut kalau sudah ada, buka setting pada backend yang sudah di import pilih port lalu ubah path menjadi \\.\pipe\com_1, agar dapat terhubung.

7. Yang kedua buka setting pada development system yang sudah diimport, lalu pilih juga port dan ganti seperti poin 6, berbeda dengan backend pada development system pilih setting pada menu sistem naikkan prosesor atau cpu-nya menjadi 2 dan pilih menu tampilan lalu aktifkan atau centang pada bagian 3d.

8. Jika sudah di setting semuanya, jalankan keduanya

9. Selesai
