## Dasar Teori

Pada praktikum ini dilakukan eksplorasi terhadap Xinu OS yang dijalankan melalui Oracle VM VirtualBox untuk mempelajari dasar - dasar sistem operasi. 

Kegiatan ini bertujuan agar dapat bisa memahami cara kerja sistem operasi dan bagaimana cara sistem mengelola sumber daya komputer

1.Selanjutnya, uraikan serta jelaskan fungsi dari setiap perintah yang terdapat di dalam Xinu!
jawaban :
argecho: Menampilkan kembali argumen yang diketik setelah perintah ini (echo argumen).
arp: Menampilkan tabel ARP (Address Resolution Protocol), yaitu daftar alamat IP dan MAC yang dikenal sistem.
cat: Menampilkan isi file atau data dari perangkat tertentu.
clear: Membersihkan tampilan layar terminal Xinu.
date: Menampilkan tanggal dan waktu sistem.
devdump: Menampilkan informasi detail mengenai perangkat (device) tertentu.
echo: Menampilkan teks atau pesan yang diketik setelah perintah.
help: Menampilkan daftar semua perintah shell Xinu yang tersedia.
kill: Menghentikan proses berdasarkan ID proses (process ID).
ls: Menampilkan daftar file atau direktori yang ada di sistem Xinu.
memdump: Menampilkan isi memori sistem dalam bentuk heksadesimal (untuk debugging).
memstat: Menampilkan status penggunaan memori yang sedang aktif.
ns: Menjalankan Network Server bawaan Xinu (biasanya untuk tes jaringan).
netinfo: Menampilkan informasi konfigurasi jaringan, seperti IP address, subnet, dan gateway.
ping: Mengirim paket ICMP untuk menguji konektivitas ke alamat IP lain.
ps: Menampilkan daftar proses yang sedang berjalan di sistem Xinu.
sleep: Menunda eksekusi sistem untuk beberapa detik (delay sementara).
tee: Menampilkan output ke layar sekaligus menyimpannya ke file.
udp: Mengirimkan paket data menggunakan protokol UDP.
udpecho: Menjalankan layanan echo berbasis UDP (membalas paket UDP yang diterima).
udpserver: Menjalankan server UDP sederhana untuk menerima koneksi dan data.
uptime: Menampilkan lama waktu sistem Xinu telah berjalan sejak terakhir boot.
?: Alias dari help, digunakan untuk menampilkan daftar perintah shell.

2. Jawablah pertanyaan-pertanyaan berikut ini: 
    - Berapa jumlah perintah pada Xinu?
      jawab: Umumnya sekitar 20-30 perintah
    - Sebutkan 2 perintah yang mempunyai fungsi yang sama!
      jawab: memstat dan freemem, sama-sama menampilkan informasi memori
    - Berapa IP address Xinu?
      jawab: Kalau pakai NAT default VirtualBox adalah 10.0.3.15   
    - Perintah apa yang digunakan untuk mengetahui IP address?
      jawab: ipconfig
    - Berapa IP DNS server yang digunakan oleh Xinu?
      jawab: Biasanya 8.8.8.8 (Google DNS) atau mengikuti settingan jaringan
    - Terdapat berapa proses yang sedang berjalan pada Xinu?
      jawab: Bisa dilihat dengan ketik ps, jumlahnya biasanya 5-10 proses
    - Proses apa yang mempunyai prioritas paling rendah?
      jawab: idle process 
    - Proses apa yang mempunyai ukuran paling besar?
      jawab: main process 
    - Proses apa yang berada dalam state current?
      jawab: main atau proses aktif saat itu
    - Proses apa yang berada dalam state suspend?
      jawab: proses yang di suspend manual atau menunggu event 
    - Berapa PID (Process ID) dari Main process?
      jawab: PID = 0 atau 1
